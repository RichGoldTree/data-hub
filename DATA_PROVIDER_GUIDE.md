# Data Provider Guide

## 1. Required Data Format
- File format: CSV
- Encoding: UTF-8 or CP949

## 2. Required Metadata
- Dataset name
- Data provider
- License

## 3. Validation Rules
- CSV must be readable by the system
- Invalid files will be rejected

## 4. Communication Flow
1. Provider prepares data following the guide
2. Provider uploads data via web interface
3. Data Hub validates and registers the dataset
